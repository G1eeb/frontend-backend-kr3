# PWA Notes

Прогрессивное веб-приложение для создания заметок и напоминаний с push-уведомлениями, офлайн-режимом и возможностью установки на устройство.

## Описание

PWA Notes — это современное веб-приложение, которое сочетает в себе функциональность заметок и напоминаний. Приложение работает полностью офлайн, поддерживает push-уведомления с возможностью откладывания, устанавливается на домашний экран как нативное приложение и синхронизирует напоминания в реальном времени через WebSocket.

### Основные возможности

- 📝 **Создание и управление заметками** — добавляйте, редактируйте и удаляйте заметки
- ⏰ **Напоминания с датой и временем** — установите напоминание на любую дату и время
- 🔔 **Push-уведомления** — получайте уведомления даже когда приложение закрыто
- ⏱️ **Откладывание напоминаний** — кнопка "Отложить на 10 секунд" в самом уведомлении
- 📱 **Установка на устройство** — работает как нативное приложение через PWA
- 📶 **Полная офлайн-работа** — доступ к заметкам без интернета
- 💬 **Реалтайм-синхронизация** — через Socket.IO

## Технологии

### Backend
- **Node.js + Express** — серверная часть и API
- **Socket.IO** — двунаправленная реалтайм-связь между клиентом и сервером
- **Web Push (VAPID)** — отправка push-уведомлений через браузер
- **HTTPS** — защищённое соединение (самоподписанный сертификат для разработки)

### Frontend
- **Vanilla JavaScript (ES6+)** — без фреймворков, чистый код
- **Service Worker** — кэширование, офлайн-режим, обработка push-уведомлений
- **Web App Manifest** — установка приложения на устройство
- **LocalStorage** — хранение заметок на клиенте
- **Notification API** — отображение системных уведомлений
- **Push API** — получение push-сообщений от сервера

### Инструменты
- **WebSockets** — для реалтайм-событий
- **Workbox стратегии** — кэширование статики и динамических запросов
- **mkcert** — генерация локальных SSL сертификатов


## Функционал

### Заметки
- ➕ Создание заметок с текстом
- ✏️ Редактирование существующих заметок (по двойному клику)
- ✅ Отметка о выполнении (через чекбокс)
- 🗑️ Удаление заметок
- 💾 Автоматическое сохранение в LocalStorage
- 🔄 Синхронизация между вкладками через Socket.IO

### Напоминания
- 📅 Выбор даты и времени для каждого напоминания
- ⏰ Автоматическое срабатывание в указанное время
- 🔔 Push-уведомление с текстом заметки
- ⏱️ Кнопка "Отложить на 10 секунд" прямо в уведомлении
- ❌ Отмена напоминания при удалении заметки
- 🗄️ Хранение активных таймеров на сервере

### Push-уведомления
- 🔔 Разрешение/запрет уведомлений при первом запуске
- 📨 Автоматическая подписка через VAPID
- ⏰ Уведомления приходят даже при закрытом приложении
- 🔄 Автоматическое обновление подписок (при истечении срока)
- 📊 Количество активных подписок отображается в консоли сервера

### Офлайн-режим
- 📦 Кэширование всех статических ресурсов при первом посещении
- 🚀 Мгновенная загрузка при повторных визитах
- 📝 Заметки доступны без интернета (хранятся в LocalStorage)
- 🔄 Автоматическая синхронизация при восстановлении соединения
- 🗑️ Умное удаление старых кэшей при обновлении версии

### Установка PWA
- 📱 Установка на домашний экран Android (Chrome, Firefox)
- 🍎 Установка на iOS (Safari, через "Добавить на экран Домой")
- 💻 Установка на Windows/Mac (Chrome, Edge)
- 🎨 Собственная иконка и splash-экран
- 🔒 Работа в отдельном окне без адресной строки
