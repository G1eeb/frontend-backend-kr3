# 📋 Мои дела — PWA приложение для заметок

Прогрессивное веб-приложение (PWA) для управления списком дел с поддержкой офлайн-режима, WebSocket синхронизации в реальном времени и push-уведомлений.

## ✨ Возможности

- ✅ Работа без интернета (офлайн-режим)
- 📱 Установка на телефон или компьютер как нативное приложение
- ⚡ Мгновенная загрузка (App Shell архитектура)
- 🔄 Синхронизация в реальном времени через WebSocket
- 🔔 Push-уведомления о новых задачах
- 💾 Автосохранение в localStorage
- 🎨 Адаптивный дизайн

## 🛠 Технологии

- **Service Worker** — кэширование и офлайн-режим
- **App Shell Architecture** — быстрая загрузка интерфейса
- **WebSocket (Socket.IO)** — двусторонняя связь в реальном времени
- **Push API + Web Push** — уведомления
- **Web App Manifest** — установка приложения
- **LocalStorage** — хранение данных
- **HTTPS** — безопасное соединение (локальный сертификат)
