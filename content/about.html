📱 О приложении

Версия 2.0.0

Это PWA приложение для управления списком дел, работающее полностью офлайн.

✨ Возможности:

  • Работа без интернета
  • Установка на телефон
  • Мгновенная загрузка (App Shell)
  • Автосохранение в localStorage
  • WebSocket синхронизация в реальном времени
  • Push-уведомления

🛠 Технологии:

  • Service Worker
  • App Shell Architecture
  • Cache API
  • WebSocket (Socket.IO)
  • Push API
  • LocalStorage
